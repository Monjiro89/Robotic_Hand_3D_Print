# Robotic_Hand_3D_Print
How to write code in Arduino IDE and Python to control a 3D printed prosthesis.
### 3D Printed Hand

This project aims to control a 3D-printed prosthetic hand using a camera motion sensor.

# Authors
@Monjiro89

##Materiales
1. *Arduino 1*
2. *Protoboard*
3. Webcam USB
4. Servomotor Micro Sg90 1.6kg Servo Motor Arduino Pic 180° (1)
5. Servomotor Mg995 15kg 180° Giro Continuo Engrane Metal (4)
6. Fuente de alimentación 5V 5A.

##Ejecución
Una vez instaladas las librerias y modulos de python, al igual que ajustar el código en base a tus conexiones con arduino, selección de camara a usar y/o fuente de alimentación, puedes ejecutar el código de hand_control.py y comenzar a probar.
